# mini-browser-app
A simple web browser application built with Node.js, Express.js, HTML, and CSS that fetches and displays webpage content based on user input.



##  Features

*  Enter any website URL
*  Fetch webpage content using server
*  Display website inside the app
*  Handles invalid URLs gracefully
*  Simple and clean user interface



## Technologies Used

* Node.js
* Express.js
* HTML5
* CSS3
* Axios



##  Project Structure

mini-browser-app/
│── server.js
│── package.json
│── package-lock.json
│── public/
│   │── index.html
│   │── style.css

---


##  Usage

1. Enter a website URL (example: `example.com`)
2. Click the **Go** button
3. The webpage will load inside the app

---

##  Limitations

Some websites (like Google, Facebook) may not load due to:

* Security restrictions
* CORS policy
* X-Frame-Options blocking
